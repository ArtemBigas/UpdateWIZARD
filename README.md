# UpdateWIZARD
Тестовое задание для Визардсофт  
Проект создан на Qt 6.9.3 под MinGW. Проект успешно собирается при при правильно настроенном Qt.  
В самом проекте, в  .pro(qmake) необходимо вручную добавить путь до установленной библиотеки QtitanComponents вместо "path_to_QTitan".  
Все изображения программа запрашивает из папки res рядом с exe файлом
