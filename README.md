# UpdateWIZARD
The project was created in Qt 6.9.3 using MinGW as an example of how to use the DevMachines component library. The project builds successfully when Qt is configured correctly.  
In the project’s .pro (qmake) file, you must manually add the path to the installed QtitanComponents library instead of "path_to_QTitan".  
All images are loaded by the application from the res folder located next to the executable file.  
   
Проект создан на Qt 6.9.3 под MinGW как пример использования библиотеки компонентов DevMachines. Проект успешно собирается при при правильно настроенном Qt.  
В самом проекте, в  .pro(qmake) необходимо вручную добавить путь до установленной библиотеки QtitanComponents вместо "path_to_QTitan".  
Все изображения программа запрашивает из папки res рядом с exe файлом
